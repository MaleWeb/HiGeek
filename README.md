## 特性

- :fire:**我的笔记**：采用OSS云储存+分布式储存+本地打包技术，再也不担心我的资料丢失了
- :100:**知识库**：采用v3.0爬虫技术，搜刮你喜欢的学习资源，创建持久访问链接
- :tophat:**协作**：基于最高2人同时协作，别问为什么，就是那么任性。
